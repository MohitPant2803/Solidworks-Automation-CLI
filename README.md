# SolidWorks AI Copilot CLI

An AI-powered, production-grade standalone desktop assistant for SolidWorks. The Copilot integrates directly with the SolidWorks COM API to translate natural language design instructions into standard CAD features, assembly configurations, mating conditions, and orthographic engineering drawings.

---

## Key Features

- **🗣️ Natural Language CAD**: Describe a part in plain English, and the Copilot builds the corresponding sketches and features.
- **⚙️ Feature Modeling**: Supports standard part modeling features:
  - Extrude Boss/Base (`create_plate`)
  - Hole Wizard / Extruded Cut (`create_hole`)
  - Fillets (`apply_fillet`)
  - Modifying Dimensions dynamically (`modify_dimension`)
  - Material Assignment (`assign_material`)
- **🧩 Assembly Automation**: Creates assemblies, inserts part/sub-assembly components at custom coordinates, and programmatically applies mates (concentric, coincident, parallel, perpendicular) using face-topology detection.
- **📐 Automated Engineering Drawings**: Instantiates drawing sheets (`.slddrw`) and automatically drops 4 orthographic views (Front, Top, Right, and Isometric) of a target model.
- **💾 Database State Tracking**: Uses an SQLite database (`database.db`) to log features and properties. Enables automatic backup checkpoints and full multi-step CAD rollback/undo functionality.
- **🎨 Premium Dark Theme CLI**: Restyles terminal outputs using `rich` Panels, styled status banners, and formatting blocks for a sleek development aesthetic.

## Getting Started

### Prerequisites

- **Windows OS** (required for SolidWorks and win32 COM binding).
- **SolidWorks installed** (supports desktop client versions).
- **Gemini API Key** (Get a free key from [Google AI Studio](https://aistudio.google.com/)).

---

## Running the Application

### 1. Standalone Executable
Run the pre-compiled executable directly from the command-line:
```powershell
# Default project
.\SolidWorksAI.exe

# Custom named project
.\SolidWorksAI.exe MyBenchVise
```

### 2. Running From Source
If running in a Python development environment:
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Launch the app:
   ```bash
   python main.py
   ```

## Command Reference

The interactive console loop supports these utility commands:

| Command | Description |
|---|---|
| `help` | Display formatting references, system statuses, and available commands. |
| `set key <api_key>` | Update the Gemini API key in `config.json` and in-memory immediately. |
| `show plan` | View the current step list of CAD commands generated by the planner. |
| `show model` | Read the active document feature tree from SolidWorks. |
| `history` | Print conversation history stored in the SQLite database. |
| `undo` | Revert the last executed CAD operation. |
| `save` | Save the active document. |
| `load <path>` | Open a `.sldprt`, `.sldasm`, or `.slddrw` document. |
| `clear` | Clear conversation history from memory. |
| `exit` / `quit` | Exit the CLI. |

## Prompting Guide

The Gemini planner interprets natural language instructions and translates them into execution scripts.

### Part Modeling
> *"Create a steel plate 120 by 120 mm with 8 mm thickness named 'base'. Then add a 12mm hole at coordinate x=10, y=10 on the face."*

### Assemblies & Mates
> *"Create a new assembly 'bench_vise'. Add the base part from 'C:\CAD\base.sldprt' and the jaw part from 'C:\CAD\jaw.sldprt'. Apply a concentric mate between them."*

### Drawings
> *"Generate a detailed drawing for the model 'C:\CAD\base.sldprt' and save it at 'C:\CAD\base_drawing.slddrw'."*
